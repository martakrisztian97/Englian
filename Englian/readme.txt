The steps to run the application:

1. Install XAMPP, if not installed. (Version: XAMPP 8.2.4)

2. Install NetBeans, if not installed. (Version: Apache NetBeans IDE 16).

3. Run NetBeans:
	- Open the project named Englian in NetBeans.

4. Add the mysql-connector-java-5.1.39-bin.jar file to the application project:
	- Right-click on the Libraries folder.
	- Click "Add JAR/Folder..." and select the mysql-connector-java-5.1.39-bin.jar file.

5. Run XAMPP:
	- Start the Apache and MySQL modules.
	- Click on the Admin button of the MySQL module to open the phpMyAdmin interface.

6. Import englian.sql database:
	- In phpMyAdmin interface click on the Import button.
	- Click on the Select file button and select the englian.sql file.

7. Run the application.
